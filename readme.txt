== Description ==

All website post link checker social network website and save shared counter..

* Jquery ajax display
* Auto added content before or after
* Admin setting pages
* Re-Generate all posts statistics
* Basic and simple themes area. Examp: {{facebook}} shared
* Cache enable or disable (0 query after cache)
* Multi site functions..
* 2 widget, greate:
 - Popular shared posts list
 - all shared counter

== Installation ==

1. Install the plugin like you always install plugins, either by uploading it via FTP or by using the "Add Plugin" function of WordPress. (Upload directory 'kuaza-post-shared-tracker' to the '/wp-content/plugins/' directory)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Activate the plugin at the plugin administration page
4. And more please look this page: (Turkish) http://kuaza.com/wordpress-eklentileri/wp-yazilarinizin-ne-kadar-paylasildigini-gosterin-facebooktwittergoogle-v-s
